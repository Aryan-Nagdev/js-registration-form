# js-registration-form
Responsive user registration form with JavaScript validation, password strength meter, and error handling.


A responsive registration form built using HTML, CSS, and JavaScript.  
This project demonstrates client-side form validation and UI feedback.

##  Features

- All fields mandatory validation
- Email format validation
- Password strength meter
- Password match check
- Phone number validation (10 digits)
- Terms & Conditions checkbox required
- Error messages below inputs
- Invalid field highlight (red border)
- Success popup message
- Responsive design

##  Technologies Used

- HTML5
- CSS3 (External stylesheet)
- JavaScript (Vanilla JS)
